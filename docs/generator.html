세계관 설정 문서게임 시작 대사도입 소품 (편지, 방송 등)

만드는 순서
1영어 학습 설정을 먼저 결정합니다 (부품 ②)
2"왜 이 방법으로만 소통해야 하는지" 세계관 이유를 만듭니다
3그 이유가 자연스러운 공간/시대/상황을 설정합니다
4위기(시간 제한)와 위화감 장치(이상한 점)를 추가합니다
5프롤로그 스크립트를 작성합니다 (게임 시작 시 읽어줄 편지/방송 등)
체크
  • 학생이 "왜 이렇게 해야 해?"라고 물으면 세계관이 답이 되는가?
  • "게임 규칙이니까"가 아니라 학생이 스스로 이유를 설명할 수 있는가?

핵심 어휘장 (40~60단어)표현 패턴 가이드게임 준비 카드

영어 표현 설계 순서
1시나리오에서 자주 등장할 상황을 정리해요 (기억 묘사, 단서 공유, 추리 등)
2각 상황에서 필요한 영어 표현 패턴을 만들어요
3표현에 필요한 핵심 어휘 40~60개를 뽑아요
4게임 준비 카드를 만들어요: 어휘장 + 표현 가이드 + 준비 퀴즈
라운드별 영어 포인트 예시
  • R1 탐색: 묘사 표현 — "I see...", "There is/are...", "It looks like..."
  • R1 토론: 기억 공유 — "I remember...", "In my card...", "I found..."
  • R2 토론: 추론 — "I think...", "Maybe...", "What if...?"
  • R3 토론: 설득 — "I believe the answer is...", "Because...", "We should..."
  • 투표: 합의 — "I agree/disagree", "Let's choose...", "My reason is..."
게임 준비 카드 구성 (박스에 동봉)
  • 어휘장 — 게임에 등장하는 핵심 단어 (뜻 + 예문)
  • 표현 가이드 — 위 패턴들을 정리한 A4 1장짜리 카드
  • 준비 퀴즈 — 짝끼리 영어 퀴즈 (게임 시작 전 15분)
⚙️ 바꿀 수 있는 부품 ③ — 소통 규칙 (⭐ 제일 중요!) 바꿀 수 있음

소통 규칙 정리규칙 설명 카드

기본 흐름

[카드 읽기] →(바꿔서 전달 = 학습!)→ [팀원이 이해] → [합쳐서 추리] → [게임 진행]

만드는 순서
1카드 정보의 형태를 결정합니다 (한국어 문장? 수식? 데이터?)
2전달하기 위해 필요한 변환 행위를 정의합니다 (= 학습 행위)
3세계관이 이 제한을 자연스럽게 설명하는지 확인합니다
핵심 체크
  • "학습 행위를 하지 않으면 게임이 멈추는가?" → 예 = 좋은 설계
  • 좋은 예: "마법이 한국어에 걸려서 영어를 써야 한다" (세계관이 규칙 설명)
  • 나쁜 예: "이 게임에서는 영어로 말하세요" (규칙만 강제)

캐릭터 시트 (인원수)장캐릭터 일러스트/이미지관계도

좋은 캐릭터의 3가지 조건
  • 나만 아는 정보 — 각 캐릭터가 다른 사람은 모르는 비밀을 갖고 있어야 해요 ("내가 말 안 하면 아무도 몰라!")
  • 숨겨진 관계 — 캐릭터끼리의 관계가 처음에는 비밀이에요 (알아가는 재미!)
  • 공감 — 학생들이 잘 아는 인물이면 좋아요 (동화, 설화, 역사 인물 등)
관계 구조 선택
  • 2쌍 (4명) — 남매+연인 등. 가장 안정적, 추천
  • 삼각 (3명) — 빠른 진행. 50분 플레이에 적합
  • 팀 (4~5명) — 역할 분담. 캠프/특별활동에 적합
캐릭터 시트 포함 내용 (머더미스터리 수준)
  • 기본 정보 — 이름, 출처, 성격 한 줄, 일러스트
  • 나만 아는 비밀 — 다른 사람은 절대 모르는 핵심 정보 (게임의 열쇠!)
  • 라운드별 공개 흐름 — R1에서 뭘 기억하고, R2에서 뭘 더 알게 되고, R3에서 전체가 드러나는 구조
  • 숨겨진 관계 — 다른 캐릭터와의 관계 (처음에는 비밀, 게임 중 발견)
  • 대표 영어 표현 — 이 캐릭터가 게임 중 자주 사용할 문장 패턴
⚙️ 바꿀 수 있는 부품 ⑤ — 반전 / 트릭 바꿀 수 있음

🔒 가짜 답이 무너지는 흐름(R1 매력적→R2 이상한데?→R3 무너짐!)은 기본 뼈대예요. 여기서는 내용만 바꿔주세요.

📦 어떻게 만드나요?
만들어야 할 것

가짜 답 설계가짜 답이 무너지는 시나리오 (R1~R3)

만드는 순서 (가짜 답이 무너지는 3단계)
1가짜 답 만들기: 처음에 "이거다!" 싶은 매력적인 가짜 해결책을 만들어요
2진짜 답 만들기: 학습을 해야만 발견할 수 있는 진짜 해결책을 만들어요
3무너지는 단서 배치:
  • R1: 함정이 매력적 + 미세한 위화감을 심어둠
  • R2: 함정 여전히 매력적 + "근데 이상한데..." 느낌 + 진짜 답의 직접적 단서 등장
  • R3: 함정 시도 시 실패 장면 제시 → "그래! 진짜 답은 이거였어!" 깨달음
꼭 기억하세요

반전에서 깨닫는 메시지가 영어 학습과 연결되어야 해요. "소통이 답이었어!", "서로의 이야기를 영어로 나눠야 했어!" 이런 식으로요.

⚙️ 바꿀 수 있는 부품 ⑥ — 카드에 적을 내용 바꿀 수 있음

카드 종류(나만 보는 카드, 다 같이 보는 단서, 힌트, 엔딩)는 항상 똑같지만, 카드에 적는 내용은 바꿀 수 있어요.

📦 어떻게 만드나요?
만들어야 할 것 (장수 계산법)

나만 보는 카드 = 캐릭터 수 × 3라운드 (예: 4×3=12장) 다 같이 보는 단서 = 라운드당 4~6장 × 3 (예: 5×3=15장) 힌트 카드 = 3단계 × 3라운드 (예: 3×3=9장) 엔딩 카드 = 엔딩 수 (예: 3장)

나만 보는 카드 — 이렇게 써요
  • R1: 흐릿하고 단편적 — 감각 위주 (냄새, 소리, 촉감). 핵심 정보는 아직 숨김
  • R2: 좀 더 구체적 — 장소, 상황이 드러남. 관계 힌트 포함
  • R3: 거의 완전한 기억 — 이름, 관계가 드러남. 이야기가 완성됨
  • 모든 카드는 그냥 읽어주면 안 되고, 바꿔서 전달해야 해요 (=여기서 학습이 일어남!)
다 같이 보는 단서 — 이렇게 나눠요
  • 라운드당 가짜 단서 1~2장 (매력적인 함정!)
  • 라운드당 진짜 힌트 1~2장 (진짜 답을 향한 복선)
  • 라운드당 분위기 카드 1장 (몰입용)
  • 학습 언어 60% / 모국어 40% 비율 추천
힌트 카드 — 이렇게 써요
  • 1단계 (넌지시): 분위기 암시 — "이 집에서 가장 중요한 건 문이 아닐지도…"
  • 2단계 (방향): 키워드 노출 — "'기억'이라는 단어가 곳곳에 등장합니다."
  • 3단계 (거의 답): 구체적 행동 안내 — "각자의 과거를 서로에게 이야기해보세요."
